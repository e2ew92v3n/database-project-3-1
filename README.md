# 3-1 데이터베이스 프로젝트

## 1. 프로젝트 설명
### 주제
온라인 쇼핑몰에서 물건을 장바구니에 담아 구매하는 기능을 데이터베이스를 통해 처리하는 방법으로 구현해보았다.

### 언어, 실행환경
Python3, SQLite3

### 구현한 기능
* 로그인
* 회원가입
* 물품 구매하기
* 장바구니에 물품 담기, 구매하기
* 구매내역 확인하기
