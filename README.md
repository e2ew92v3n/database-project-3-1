# 3-1 데이터베이스 프로젝트

## 1. 프로젝트 설명
### 주제
온라인쇼핑몰에서 물건을 구매하고 장바구니에 담는 기능을 데이터베이스 처리하는 방법을 표현하기 위해 만들어 보기로 하였다.

### 언어, 실행환경
Python, sqlite3

### 구현한 기능
* 로그인
* 회원가입
* 물품 구매하기
* 장바구니에 물품 담기, 구매하기
* 구매기록 확인하기
